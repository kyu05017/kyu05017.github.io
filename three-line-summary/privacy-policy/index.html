세줄요약 개인정보처리방침

앱 이름: 세줄요약 (Three-Line Summary)
최종 업데이트: 2026-04-01
문의 이메일: monoManager@gmail.com

세줄요약(이하 “앱”)은 음성 기록과 요약 기능을 제공하기 위해 필요한 최소한의 정보를 처리합니다. 아래 내용은 앱 사용 중 어떤 정보가 처리될 수 있는지와 사용자가 어떤 선택을 할 수 있는지를 설명합니다.

1. 처리할 수 있는 정보

  • 음성 기록을 통해 생성된 텍스트(원문)
  • 요약 결과 텍스트
  • 세션 이름, 요약 상황, 요약 방식, 사용자 편집 내용
  • 앱 언어, 테마, 요약 설정 등 설정값
  • 기기 식별값 및 서버 접근 토큰(서비스 제공 목적)

2. 정보 이용 목적

  • 음성 기록 내용을 텍스트로 표시하고 요약 결과를 생성하기 위해
  • 사용자가 생성한 히스토리와 설정을 기기에 저장하고 다시 불러오기 위해
  • 요약 요청 시 서버와 통신하고 인증 세션을 유지하기 위해
  • 사용자가 요약 결과를 검토, 편집, 공유할 수 있도록 하기 위해
  • 앱 안정성 점검 및 오류 대응을 위해

3. 저장 및 전송

일부 정보는 사용자의 기기에 로컬로 저장될 수 있습니다. 사용자가 요약 생성을 요청하면, 원문, 세션 정보, 요약 설정 등 서비스 제공에 필요한 정보가 운영 서버로 전송될 수 있습니다.

4. 보관 및 삭제

  • 사용자가 생성한 히스토리는 앱 내 로컬 저장소에 보관될 수 있습니다.
  • 사용자는 앱 내 기능을 통해 히스토리 삭제, 휴지통 이동, 복구 등을 수행할 수 있습니다.
  • 앱을 삭제하거나 앱 내 데이터 삭제 기능을 실행하면 일부 로컬 정보는 제거될 수 있습니다.

5. 외부 서비스

앱은 원칙적으로 사용자의 정보를 판매하지 않습니다.

  • 요약 서버 운영 인프라: 요약 요청 처리, 인증 세션 발급, 오류 대응
  • 플랫폼 기본 공유 기능: 사용자가 직접 선택한 외부 앱으로 텍스트를 공유할 때

6. 사용자 선택

  • 앱 내 저장된 히스토리 열람
  • 히스토리 삭제 및 복구
  • 원문 저장 여부 등 일부 설정 변경
  • 공유 전 내용 검토 및 편집
  • 앱 사용 중단 및 앱 삭제
